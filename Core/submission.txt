تمرین مدیریت هزینه‌ها با FastAPI

نام فایل: manage_cost.py

توضیح کوتاه:

- تمام endpointهای CRUD برای مدیریت هزینه‌ها (POST, GET, PUT, DELETE) در فایل manage_cost.py پیاده‌سازی شده است.
- هزینه‌ها در حافظه (دیکشنری) ذخیره می‌شوند و شامل id، description و amount هستند.
- خروجی‌ها با JSONResponse و کد وضعیت HTTP مناسب برگردانده می‌شوند.

توضیح مهم:

- به دلیل عدم دسترسی به اینترنت بین‌الملل، امکان قرار دادن پروژه روی GitHub وجود نداشت.
